# 🚀 Web Projem - AI Test Platformu

Bu proje, Yapay Zekâ (AI) modellerini ve entegrasyonlarını test etmek amacıyla geliştirilmiş web tabanlı bir uygulamadır. 

Proje, **GitHub Pages** üzerinde doğrudan barındırılabilecek ve çalıştırılabilecek şekilde statik web teknolojileriyle optimize edilmiştir.

## 🌟 Özellikler
- **AI Entegrasyonu & Test:** Farklı yapay zekâ senaryolarını ve API'lerini test etmek için hazır altyapı.
- **Hızlı Dağıtım:** Sunucu kurulumu gerektirmeden doğrudan GitHub Pages üzerinde çalışabilme.
- **Kullanıcı Dostu Arayüz:** Modern, temiz ve hızlı yanıt veren tasarım.

## 🛠️ Kurulum ve Canlıya Alma (GitHub Pages)

Bu projeyi kendi GitHub Pages ortamınızda yayına almak için aşağıdaki adımları takip edebilirsiniz:

1. **Projeyi Fork Edin:** Sağ üst köşedeki `Fork` butonuna tıklayarak projeyi kendi hesabınıza kopyalayın.
2. **GitHub Pages'i Aktif Edin:**
   - Deponuzun (Repository) **Settings (Ayarlar)** sekmesine gidin.
   - Sol menüden **Pages** seçeneğine tıklayın.
   - **Build and deployment** bölümünde, Source kısmını `Deploy from a branch` olarak seçin.
   - Branch olarak `main` (veya projenizin ana dalını) ve klasör olarak `/ root` seçip **Save** butonuna tıklayın.
3. **Canlı Link:** Birkaç dakika içinde projeniz `https://<kullanici-adiniz>.github.io/webprojem/` adresinde yayında olacaktır.
